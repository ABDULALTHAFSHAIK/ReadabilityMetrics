Install curl (in Ubuntu):

sudo apt-get install curl

1. Testing connection:

curl -i -X GET http://localhost:8888/readability/rest/

2. Testing insert:

curl -i -X POST -H 'Content-Type: text/plain' -d "This is an example test for Readability." http://localhost:8888/readability/rest/insertText

This returns the id to be used in the following commands.

3. Testing getting: 

curl -i -X GET http://localhost:8888/readability/rest/getText/<id>

4. Testing updating:

curl -i -X PUT -H 'Content-Type: text/plain' -d "This is an updated text for Readability." http://localhost:8888/readability/rest/updateText/<id>

5. Testing delete:

curl -i -X DELETE http://localhost:8888/readability/rest/deleteText/<id>

6. Testing all metrics in HTML:

curl -i -X GET http://localhost:8888/readability/rest/getHtmlMetrics/<id>/
